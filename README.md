# 🚀 API de Cadastro de Usuários

Sistema completo de cadastro de usuários com validação de CPF/CNPJ, persistência em banco de dados e integração com API externa.

## 🧠 Sobre o projeto

Este projeto foi desenvolvido com foco em prática de backend utilizando Python, incluindo:

- Construção de API REST
- Integração com banco de dados
- Validação de dados
- Consumo de API externa
- Integração com frontend simples

---

## ⚙️ Tecnologias utilizadas

- Python
- FastAPI
- SQLite
- SQLAlchemy
- HTML, CSS, JavaScript

---

## 🔌 Funcionalidades

- ✔️ Criar usuário
- ✔️ Listar usuários
- ✔️ Atualizar usuário
- ✔️ Deletar usuário
- ✔️ Validação de CPF/CNPJ
- ✔️ Integração com API externa (dados adicionais)

## 🖥️ Como rodar o projeto

### 1. Clonar repositório

```bash
git clone https://github.com/seu-usuario/api-cadastro.git
```

### 2. Entrar na pasta

```bash
cd api-cadastro
```

### 3. Criar ambiente virtual

```bash
python -m venv venv
```

### 4. Ativar ambiente

```bash
venv\Scripts\activate
```

### 5. Instalar dependências

```bash
pip install -r requirements.txt
```

### 6. Rodar servidor

```bash
uvicorn app.main:app --reload
```
